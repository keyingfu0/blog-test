bundle spliting 和 code spliting

分别是:

  • splitChunks
  • 动态导入import()

配置为splitChunks: initial,这样所有同样的动态导入会打包为一起,所有相同的非动态导入会打包到一起,只有那种既有动态又有非动态的会被分开(所以要避免这样)
